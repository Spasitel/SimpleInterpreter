Особенности работы:
    * Последовательности нельзя заключать в (). Пример: ({2 , 3}) вернет синтаксическую ошибку.
    * Знак "-" перед выражением имеет приоритет над остальными операциями. Пример: -2^2 = 4.
    * Внутри лябмд нельзя использовать параметры внешних лямбд. Пример: reduce({1, 2}, 1 , x y -> reduce({2, 3}, 3, m n -> x)) вернет ошибку использования х.
